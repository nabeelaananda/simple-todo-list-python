# simple-todo-list-python
My second Python project: a terminal-based To-Do list demonstrating loops, conditionals, and error handling.

A lightweight, interactive Command Line Interface (CLI) application built with Python to manage daily tasks efficiently. 

## ✨ Features
* **Add Tasks:** Easily append new items to your to-do list.
* **Remove Tasks:** Delete specific tasks using their dynamic index number.
* **Input Validation:** Built-in error handling to prevent crashes from invalid user inputs (e.g., entering letters instead of numbers).
* **Empty List Handling:** Gracefully notifies the user when the list is empty and prevents invalid deletion attempts.

## 🚀 How to Run
1. Ensure you have Python installed on your system.
2. Clone this repository or download the `todo.py` file.
3. Open your terminal or command prompt.
4. Run the script using the following command:
   ```bash
   python todo.py
